## JSON Parser

This is my second C++ project, and I built this because I wanted to understand the syntax of C++ better.

This was very fun to build as I got to model this parser as a finite state machine, so I could see exactly how the states interacted.

I also got to implement a stack datatype for the nested arrays.

If you want to experiment with it yourself with different json inputs, go to the [Github Release](https://github.com/VulcanWM/json-parser/releases/tag/v1.0.0) and follow the instructions.

Thanks for reading, and I'd love any C++ critique.
